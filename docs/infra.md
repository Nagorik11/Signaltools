# Infraestructura y Configuración

Detalles sobre la construcción, dependencias y organización del repositorio.

## Configuración del Proyecto
- `pyproject.toml`: Define los metadatos del proyecto, requisitos del sistema y herramientas de construcción (build system). Indica que el proyecto sigue el estándar de paquete moderno de Python.
- `README.md` (Raíz): Contiene la descripción general del proyecto en inglés, instrucciones de instalación y uso rápido.

## Otros Directorios
- `build/`: Carpeta temporal generada durante la instalación o creación de paquetes.
- `output/`: Directorio por defecto donde se guardan los resultados de los análisis y reportes forenses si no se especifica otra ruta.
- `docs/`: Documentación original (si existe) que sirve de base para este manual extendido.
